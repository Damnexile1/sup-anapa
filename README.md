# SUP Anapa

Веб-сервис для бронирования SUP-прогулок в Анапе.

## Возможности

- 🏄 Бронирование SUP-прогулок онлайн
- 🌤️ Прогноз погоды при бронировании (температура воздуха, воды, ветер)
- 👨‍🏫 Выбор инструктора
- 📅 Гибкое расписание слотов
- 🔐 Административная панель для управления
- 📱 Уведомления администратору о новых бронированиях

## Технологии

- **Backend**: Go 1.21+
- **Router**: Chi
- **Database**: PostgreSQL 15+
- **Frontend**: Go templates + htmx + Tailwind CSS
- **Deployment**: Docker + Docker Compose

## Быстрый старт

### Требования

- Docker и Docker Compose
- Make

**Важно:** Ничего не нужно устанавливать на хост-машину! Все инструменты работают внутри Docker.

### Установка (одна команда!)

```bash
# Создать .env файл
make setup

# Запустить всё (Docker + БД + миграции)
make deploy-local
```

Приложение будет доступно на http://localhost:8080

### Продакшн деплой (одна команда!)

```bash
make deploy-prod
```

### Все доступные команды

```bash
make help
```

## Структура проекта

```

├── cmd/server/          # Точка входа приложения
├── internal/
│   ├── config/         # Конфигурация
│   ├── handlers/       # HTTP handlers
│   ├── middleware/     # Middleware (auth, logging)
```

## API Endpoints

### Публичные страницы
- `GET /` - Главная страница
- `GET /booking` - Страница бронирования
- `POST /booking` - Создание бронирования

### Административная панель
- `GET /admin/login` - Вход в админку
- `GET /admin` - Dashboard
- `GET /admin/instructors` - Управление инструкторами
- `GET /admin/slots` - Управление слотами
- `GET /admin/bookings` - Просмотр бронирований
